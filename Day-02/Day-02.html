Semantic Tags

Semantic tags are HTML elements that clearly describe their meaning and purpose in the webpage.
They make the structure of your page more readable for developers, browsers, and search engines.

Some examples of semantic tags are header, navbar, main, footer, section etc.

Block and Inline Elements

Block Elements :

"Block elements are elements that always start on a new line and automatically take up the full width of their container."

Some example of the block elements are div, p, h1 to h6, section etc.

Inline Elements :

Inline elements are elements that do not start on a new line and only take up as much width as their content requires.

Some Examples of the inline elements are span, a, b, i, img, label, input etc.
